## ✨ Features

- **Custom Cursor** — gold dot + trailing ring with mix-blend-mode difference effect
- **Flip Card Hero** — hover to scan & reveal profile photo, click to flip back
- **Animated Timeline** — bidirectional scroll-triggered experience section with collapsible entries
- **Spotlight Effect** — radial gradient follows mouse across the page
- **Particle System** — floating ambient particles in the hero section
- **Scroll Fade-ins** — IntersectionObserver-powered section reveals
- **Terminal About** — styled terminal block with blinking cursor
- **Fully Responsive** — mobile-first, works on all screen sizes
- **Resume Download** — one-click PDF download from the navbar
